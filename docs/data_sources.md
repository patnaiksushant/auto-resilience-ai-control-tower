# Data Sources

This project supports seven Kaggle source families:

1. Automotive supply chain: maps to suppliers, components, purchase orders and inventory.
2. General supply-chain analytics: maps to order, demand, inventory and cost signals.
3. Logistics operations: maps to shipment, carrier, transit and freight cost signals.
4. Smart logistics: maps to logistics events and route delay features.
5. Predictive maintenance: maps to production and machine maintenance records.
6. IoT smart manufacturing: maps to real-time machine sensor streams.
7. Manufacturing defects: maps to quality, defect type, defect rate and process metrics.

The app standardizes all datasets into canonical tables under app/data/processed.
