# Advanced ML and AI Methods

## Risk scoring
Weighted baseline score plus ML-ready feature frame using supplier rating, country risk, geopolitical dependency, delay history, capacity and ESG.

## Late delivery prediction
Random Forest, Gradient Boosting and Logistic Regression classify whether a PO will be late.

## Demand forecasting
Lag features, rolling means and tree-based regressors estimate next-period SKU demand.

## Anomaly detection
Isolation Forest can be applied to supplier-risk features and IoT sensor features.

## Optimization and recommendation
Alternate supplier ranking uses a weighted score across risk, capacity, lead time, supplier rating and ESG.

## Manufacturing analytics
OEE, Cp, Cpk, defect trend and energy-per-unit KPIs support shop-floor monitoring.

## AI action recommendation
Current version uses rule-based recommendations. Production extension can add a RAG/GenAI copilot over SOPs, supplier contracts, incident logs and meeting notes.
